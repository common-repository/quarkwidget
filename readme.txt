A simple widget for placing a quarkcoin donation button on your website!

== Description ==

QuarkWidget -a donation widget that Bloggers, News sites, Campaign sites, Charities etc can easily put on their websites in order to Quickly receive Quark coins from their readers.

This plugin connects to http://qrk.blockr.io to grab address balances and number of transactions. 

== Installation ==

1. Upload the folder `qrkdonation` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget to your site via the 'Widgets' menu in WordPress

== Frequently Asked Questions ==

= Q. How can I change the quark donation address =

A. Click on 'Settings' and click on the Quark Coin Donation Button link.
